## Guia de Uso das Referencias

### Para Sprite ANDRADE (NPC Inutil)
- Consultar: Andre Guedes Twitter + IMDB (Zumbis em Brasilia 2018)
- Focar em: Rosto redondo inexpressivo, olhos perdidos, terno largo, mochila escolar
- Tom: Patetismo, invisibilidade, irrelevancia total

### Para Sprite TAXADD (Mini-Boss)
- Consultar: Know Your Meme + Poder360 compilacao + Times Square
- Focar em: Oculos de vilao, calculadora gigante, cifras flutuantes, terno com estampa de dinheiro
- Tom: Vilao compulsivo, ameaca fiscal, sorriso sinistro

### Para Sprites de ATTACK
- Consultar: Tenor GIF + TikTok (movimento de taxar)
- Focar em: Calculadora disparando cifras, carimbo "TAXADO", maos enormes agarrando

### Para Sprites SPECIAL
- Taxa Infinita: Aura verde-dinheiro drenando tudo ao redor
- Taxacao Universal: TUDO no mapa com selo "TAXADO", ate a interface
- Transformacao Andrade→Taxadd: Transicao de NPC pacifico para mini-boss

### Para SKINS / VARIANTES
- Andrade: Andre Guedes 2018 (pacifico, inutil)
- Taxadd: Compilacao memes 2024 (vilao fiscal)
- Taxador do Futuro: Exterminador + impostos
- Walter Taxagrande: Walter White + taxas

### Para DEATH Animation
- Consultar: Memes gerais + Andre Guedes

## Notas Importantes

1. **Andre Guedes e a referencia PRIMARIA** de estilo - todo sprite deve parecer saido do universo dele
2. DUAS sprite sheets completamente separadas: Andrade (NPC) e Taxadd (Mini-Boss)
3. Andrade = patetismo puro, olhos de cachorro perdido, invisivel
4. Taxadd = ameaca fiscal encarnada, cifras flutuantes SEMPRE presentes
5. As maos do Taxadd sao DESPROPORCIONALMENTE GRANDES (para agarrar dinheiro)
6. Os bolsos TRANSBORDAM moedas em TODOS os frames
7. O carimbo "TAXADO" deve aparecer como efeito visual nos alvos atingidos
8. Memes reais servem como referencia de CONCEITO para skins, nao de estilo artistico
