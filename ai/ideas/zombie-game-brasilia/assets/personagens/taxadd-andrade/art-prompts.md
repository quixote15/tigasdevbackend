## NOTAS PARA GERACAO

1. **SEMPRE incluir style tags globais** em cada prompt
2. **SEMPRE incluir negative prompts** para evitar estilos indesejaveis
3. Para Stable Diffusion: usar CFG 7-9, steps 30-50, sampler DPM++ 2M Karras
4. Para DALL-E: prefixar com "pixel art game sprite" para melhor resultado
5. Para Midjourney: adicionar --style raw --no smooth,realistic --ar conforme aspecto
6. Gerar em RESOLUCAO ALTA (512x512 minimo) e depois reduzir para 64x64 com nearest-neighbor (sem anti-aliasing)
7. As maos ENORMES do Taxadd sao o elemento mais critico - se nao ficarem desproporcionais, REGENERAR
8. As cifras $ devem ser SEMPRE visiveis mesmo em thumbnail - alto contraste obrigatorio
9. Andrade deve ser INSTANTANEAMENTE diferente do Taxadd - se confundir, REGENERAR
10. Testar TODOS os sprites contra fundo laranja-sangue (ceu do jogo) para garantir visibilidade
