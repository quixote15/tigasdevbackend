# BOLSOLULA (Fusao) — Art Prompts for Image Generation

## General Style Guide (Apply to ALL prompts)

### Base Style Prefix (prepend to every prompt)
```
pixel art, 96x96 sprite, top-down slightly isometric perspective, underground comix style, Robert Crumb influence, Brazilian political cartoon by Andre Guedes, grotesque caricature, thick irregular black outlines 3-4px, dirty saturated colors, paper texture overlay, cross-hatching shadows, NOT pretty, NOT polished, hand-drawn rage aesthetic, body horror comedy, B-movie monster,
```

### Negative Prompts (append to every prompt)
```
--negative_prompt "photorealistic, 3D render, smooth shading, gradient, clean lines, symmetrical, beautiful, pretty, anime, manga, chibi, cute, kawaii, soft colors, pastel, digital art, vector art, flat design, minimalist, modern, professional, polished, refined, elegant, graceful, pure white background, solid color background, watermark, signature, text overlay, UI elements, frame border, high resolution photograph, realistic proportions, anatomically correct, flattering, heroic pose, noble, dignified, respectful"
```

- Lula Red: `#8B1A1A` (dirty crimson)
- Bolsonaro Green: `#2E5A1C` (military olive)
- Bolsonaro Orange (2026 prison): `#CC5500`
- Fusion Seam: `#5C2020` + `#8B3030` (exposed flesh)
- Lula Skin: `#B07850` (weathered tan)
- Bolsonaro Skin: `#D4A574` (pale)
- Outline: `#1A1A18`
