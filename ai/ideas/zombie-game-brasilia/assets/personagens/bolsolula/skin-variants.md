# BOLSOLULA (Fusao) — Skin Variants

## Overview

All skins maintain the core BOLSOLULA anatomy:
- Vertically split body (left = Lula, right = Bolsonaro)
- 4 arms (2 per side)
- 2 fused faces in one skull
- Fusion seam down the center with stitches and Frankenstein bolt
- Caneta BIC Gigante weapon (reskinned per variant)
- 96x96px sprite dimensions

Each skin re-textures BOTH halves. The political duality and grotesque fusion aesthetic must be preserved in every variant.

## SKIN 7: PIXEL GLITCH

### Name: "Bug do Sistema"
### Unlock: Kill BolsoLula while it's mid-Split (Fusao Reversa) — timing-critical

### Description
A corrupted digital version. The sprite is intentionally "broken" — misaligned pixels, color palette glitches, scanlines. Represents the idea that the BolsoLula fusion is a bug in reality itself.

### Visual Treatment
| Element | Detail |
|---|---|
| Pixel displacement | Random 2x2px blocks of the sprite are offset 1-2px from correct position. Different blocks shift each frame. |
| Color glitch | Random color channel shifts: R channel slides 1px left, G channel slides 1px right, creating RGB fringing effect on edges |
| Scanlines | Horizontal 1px black lines at 50% opacity every 4px (CRT effect) |
| Palette corruption | Every 3rd frame, a random 8x8px block of the sprite swaps to the WRONG character's palette (Lula block gets Bolsonaro colors, vice versa) |
| Static noise | 5% of pixels per frame are replaced with random noise (`#FFFFFF` or `#1A1A18`, random) |
| Missing rows | 2-3 horizontal pixel rows per frame are black (missing data) |

### Fusion Seam
| Element | Detail |
|---|---|
| Seam | The seam IS the glitch epicenter — the corruption radiates outward from it |
| Instead of stitches | Pixel artifact patterns: repeating 1010 binary pixel patterns along seam |
| Instead of bolt | A "missing texture" checkerboard square (magenta `#FF00FF` + black, 4x4px) |

### Special Effects
- All particles are pixelated/quantized to 2x2px blocks
- Sound effects have digital distortion/bitcrushing
- Screen tearing effect on specials (horizontal offset of top/bottom screen halves)

### Weapon
| Element | Detail |
|---|---|
| Caneta BIC | A CURSOR ARROW — the system's attempt to render the pen. Glitches between pen and cursor shape. |

## SKIN IMPLEMENTATION NOTES

### Phaser 3 Skin System
```javascript
// Each skin is a separate sprite sheet with identical frame layout
// Loaded on demand — only default + 2026 are pre-loaded

const BOLSOLULA_SKINS = {
  'normal':       { atlas: 'boss_bolsolula',            loaded: true },
  '2026':         { atlas: 'boss_bolsolula_2026',       loaded: true },
  'zombie':       { atlas: 'boss_bolsolula_zombie',     loaded: false },
  'skeleton':     { atlas: 'boss_bolsolula_skeleton',   loaded: false },
  'debate':       { atlas: 'boss_bolsolula_debate',     loaded: false },
  'carnaval':     { atlas: 'boss_bolsolula_carnaval',   loaded: false },
  'embalsamado':  { atlas: 'boss_bolsolula_embalsamado',loaded: false },
  'glitch':       { atlas: 'boss_bolsolula_glitch',     loaded: false },
  'shadow':       { atlas: 'boss_bolsolula_shadow',     loaded: false }
};

// Skin swap: 
// 1. Load atlas if not loaded (async)
// 2. Swap texture key on sprite
// 3. All animations remain identical (same frame indices)
// 4. Apply skin-specific particle overrides
// 5. Apply skin-specific sound overrides (if any)
```

### Shared Across All Skins
- Frame count and layout (identical sprite sheet structure)
- Animation timings
- Hitboxes and collision shapes
- Gameplay mechanics (damage, skills, AI)
- Fusion/Split sequences

### Skin-Specific Overrides (per skin)
- Color palette (the main difference)
- Particle effect colors
- Specific additional particles (maggots for zombie, confetti for carnaval, etc.)
- Sound overlays (bone rattle for skeleton, samba for carnaval, bitcrush for glitch)
- Death animation variant (skeleton scatters, shadow dissipates, etc.)
- Ambient effects (darkness for shadow, glass for embalsamado)
