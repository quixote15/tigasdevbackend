## Guia de Uso das Referencias

### Para Sprite de IDLE
- Consultar: Andre Guedes TikTok + Instagram
- Focar em: Careca brilhando, postura ereta, martelo apoiado no ombro

### Para Sprite de WALK
- Consultar: Andre Guedes Facebook (movimento)
- Focar em: Marcha pesada, martelo arrastando, intimidacao

### Para Sprite de ATTACK
- Consultar: Meme censura + Andre Guedes TikTok
- Focar em: Martelo descendo, "CENSURADO" aparecendo, impacto

### Para Sprites SPECIAL
- Censura Monocratica: Meme "censura a si mesmo" + PCO
- Xandaquistao: PCO "Fica X fora Xandao" + memes gerais
- Apagao Digital: Memes volta do X + Andre Guedes

### Para SKINS
- Normal: Andre Guedes (todas as refs)
- Xandaquistao: PCO + memes gerais
- Sancionado: Memes sancoes EUA + Magnitsky
- Mao TSE Tung: Adaptar toga com estetica comunista
- Dormindo: Memes "aproveitando que o Xandao ta dormindo"

### Para DEATH Animation
- Consultar: Sandman/Foucault (dramaticidade)
- Focar em: Toga se desfazendo, careca apagando, papeis voando

## Notas Importantes

1. **Andre Guedes e a referencia PRIMARIA** - todo art asset deve parecer que saiu do estilo dele
2. Os memes servem como referencia de POSES e EXPRESSOES, nao de estilo artistico
3. Sempre exagerar: careca MAIS brilhante, biceps MAIS grandes, martelo MAIS desproporcional
4. A toga deve parecer SUJA e AMARROTADA, nunca limpa e passada
5. Expressao padrao: RAIVA. Variacao: RAIVA MAIS INTENSA
