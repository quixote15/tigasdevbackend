# Nikolas Ferreira (O Influencer-Politico) - Audio Script

## Visao Geral
Nikolas fala como um INFLUENCER DIGITAL: rapido, energetico, provocador. Tom jovem, nasalado levemente, com a cadencia de quem grava stories/TikTok. Tudo e "conteudo". Cada frase e calculada para engajar.

**Voz base:** Masculina jovem (20-30 anos), timbre medio-agudo, velocidade acima da media, energia alta constante. Leve sotaque mineiro que ele tenta esconder com sotaque "neutro de internet".

**Estilo de fala:** Staccato, frases curtas, ritmo de rede social. Como se cada fala tivesse um corte de edicao no final.

---

## BORDOES PRINCIPAIS

### 1. "Isso aqui vai viralizar!"
- **Contexto:** Quando encontra o jogador / inicio de combate
- **Entonacao:** Excitacao genuina, voz subindo no "viralizar", quase gritando
- **Timing:** 1.2s total
  - "Isso aqui" (0.4s, tom medio, rapido)
  - "vai" (0.15s, pausa dramatica minima antes)
  - "VIRALIZAR!" (0.65s, tom sobe, enfase MAXIMA, quase quebrando a voz)
- **Volume:** 85% -> 100% (crescendo)
- **Trigger:** Proximidade do jogador (raio 80px) OU inicio de ataque
- **Cooldown:** 15s entre repeticoes
- **Prioridade:** Alta

### 2. "Grava, grava, grava!"
- **Contexto:** Durante ataque (Flash Attack)
- **Entonacao:** Urgencia frenetica, como se estivesse perdendo O momento
- **Timing:** 0.9s total
  - "Grava," (0.25s, rapido, tom medio)
  - "grava," (0.25s, mais rapido, tom subindo)
  - "GRAVA!" (0.4s, GRITADO, tom no maximo)
- **Volume:** 70% -> 85% -> 100% (escalada)
- **Trigger:** Frame 1 do ataque (momento do flash)
- **Cooldown:** 8s
- **Prioridade:** Maxima (durante ataque)

### 3. "O algoritmo nunca mente!"
- **Contexto:** Idle prolongado / quando views flutuantes aumentam
- **Entonacao:** Convicção falsa, tom de quem repete um mantra, levemente psicotico
- **Timing:** 1.4s total
  - "O algoritmo" (0.5s, tom medio, cadencia de aula/palestra)
  - "NUNCA" (0.3s, enfase dura, tom desce)
  - "mente!" (0.6s, tom sobe de novo, sorriso na voz)
- **Volume:** 75% constante
- **Trigger:** Idle por mais de 10s OU numeros de views atingem milestone
- **Cooldown:** 20s
- **Prioridade:** Baixa

### 4. "Voces viram meu video do PIX?"
- **Contexto:** Interacao casual / encontro nao-hostil
- **Entonacao:** Conversa casual mas calculada, como se TUDO fosse sobre engagement
- **Timing:** 1.5s total
  - "Voces viram" (0.5s, casual, tom de conversa)
  - "meu video" (0.4s, enfase sutil no "meu")
  - "do PIX?" (0.6s, tom sobe na pergunta, expectativa)
- **Volume:** 70%
- **Trigger:** Jogador proximo mas nao em combate
- **Cooldown:** 30s
- **Prioridade:** Baixa

## MIXAGEM E ESPACIALIZACAO

### Volume Hierarchy
1. **Bordoes de ataque/special:** 100% (prioridade maxima)
2. **Bordoes de reacao (hit/death):** 85-90%
3. **Bordoes de encontro:** 75-85%
4. **Bordoes idle:** 55-70%
5. **SFX celular/ring light:** 40-60% (background)

### Espacializacao (Phaser 3)
- Todas as falas sao spatialized (volume cai com distancia)
- Raio maximo de audicao: 200px
- Atenuacao: linear
- Pan: baseado na posicao X relativa a camera

```javascript
// Exemplo de espacializacao
const voice = this.sound.add('nikolas_viralizar', {
    volume: 0.85,
    detune: 0,
    rate: 1.0
});
// Ajustar volume baseado em distancia
const dist = Phaser.Math.Distance.Between(nikolas.x, nikolas.y, player.x, player.y);
const vol = Math.max(0, 1 - (dist / 200));
voice.setVolume(vol * 0.85);
```

### Efeitos de Processamento
- **Todas as falas:** Leve compressao (ratio 3:1) para manter presenca
- **Falas de morte:** Reverb crescente (room size 0.3 -> 0.8) + lowpass filter gradual
- **Falas de special:** Leve distorcao (overdrive 10%) para energia extra
- **Idle falas:** Filtro de "audio de celular" (bandpass 300Hz-3kHz) quando a camera esta longe
