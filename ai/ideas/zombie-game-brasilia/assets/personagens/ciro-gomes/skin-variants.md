# Ciro Gomes - Variantes de Skin

## VISAO GERAL

Ciro Gomes possui 5 variantes de skin. Todas manteem a silhueta base (cabecao arrogante, terno, veias no pescoco) mas alteram paleta de cores e detalhes visuais.

---

## SKIN 1: NORMAL (2026) — Skin Padrao

### Conceito
O eterno candidato. Terno impecavel, cabelo perfeito, raiva contida. O mais arrumado do elenco num cenario apocaliptico. O contraste entre a aparencia polida e o mundo destruido E a piada.

## SKIN 2: ZOMBIE (Infectado)

### Conceito
Ciro virou zumbi MAS continua arrogante. O terno esta rasgado mas ele TENTA mante-lo arrumado. Ainda gesticula e "discursa" mas so saem grunhidos. A Cirocracia virou literalmente um miasma toxico. Continua xingando, mas os xingamentos sao grunhidos ininteligiveis.

### Detalhes Especificos
- Terno: rasgado no ombro esquerdo, mas a lapela direita ainda esta impecavel (ele tenta)
- Mandibula: ligeiramente deslocada mas ainda tenta xingar

- Cabelo: finalmente bagunçado — mas ele passa a mao tentando arrumar (em vao)

- Walk: mancando mas rapido, braco direito rigido

- Xingamento: grunhido sonoro que atrai MAIS zumbis (nao stuna)

## SKIN 3: PANICO SEM RIVOTRIL (Rage Mode)

### Conceito
Skin ativada quando a barra de Rivotril chega a 0%. NAO e uma skin selecionavel — e um estado AUTOMATICO. Ciro sem remedio. Vermelho berrante, veias saltando como tentaculos, suor escorrendo, olhos injetados.

### Paleta de Cores

| Elemento             | Hex       | Notas                                   |
|---------------------|-----------|------------------------------------------|
| Terno               | #1B2A4A  | Mesmo, mas AMASSADO, desalinhado         |
| Gravata             | #CC0000  | Mais vermelha, solta, torcida            |
| Camisa              | #E0D0D0  | Manchada de suor                         |
| Pele                | #FF2222  | VERMELHO BERRANTE constante              |
| Veias               | #990099  | Roxo NEON, pulsando, 3x tamanho normal  |
| Cabelo              | #8C8C8C  | Arrepiado, fios em pe                    |
| Olhos               | #FF0000  | Injetados de sangue                      |
| Suor                | #87CEEB  | Gotas constantes                         |
| Vapor               | #FF6347  | Saindo das orelhas                       |
| Rivotril frasco     | #FF6B00  | Quebrado, rachado, vazio                 |
| Contorno            | #000000  | Tremulo, instavel                        |

### Detalhes Especificos
- Rosto: VERMELHO CONSTANTE, pulsa entre #FF2222 e #FF0000 a cada frame
- Veias: saltam do pescoco como TENTACULOS, se estendem pela testa e bracos
- Terno: amassado, botao aberto, camisa pra fora
- Gravata: torcida como corda
- Olhos: completamente vermelhos, pupilas dilatadas
- Suor: gotejando CONSTANTEMENTE (particulas a cada frame)
- Vapor: sai das orelhas (2 particulas constantes)
- Cabelo: arrepiado pela primeira vez (normalmente perfeito)
- Frasco: segurado com forca, rachado, vazio
- Tremor: TODOS os sprites tremem 3px aleatorio
- Contornos: linhas TREMEM (nao sao retas, sao irregulares)

### Transicao Visual
- Normal -> Panico: gradiente de 0.5s
  - Pele: #D4956A -> #CC3333 -> #FF2222
  - Veias: 1px -> 2px -> 3px largura
  - Tremor: 0px -> 1px -> 3px
  - Cabelo: penteado -> levemente bagunçado -> arrepiado
  - Olhos: castanho -> injetados -> full vermelho

## NOTAS DE IMPLEMENTACAO

### Troca de Skins
- Skins selecionaveis: Normal, Zombie, Astral, Classico 2018 (menu de selecao)
- Skin Panico: NAO selecionavel, ativada automaticamente por gameplay
- Transicao entre skins normais: corte direto (sem blend)
- Transicao Normal -> Panico: gradiente de 0.5s
- Transicao Panico -> Normal: gradiente de 1.0s (volta mais devagar)

### Sprite Sheets por Skin
Cada skin precisa de suas proprias sprite sheets para TODAS as animacoes:
- 11 animacoes x 5 skins = 55 sprite sheets individuais
- OU: usar palette swap programatico (recomendado pra economizar espaco no atlas)
- Palette swap: trocar cores via shader/lookup table em runtime

### Palette Swap Sugerido
Manter sprite sheet unica com cores "marcadoras" e trocar em runtime:
- Cor marcadora pele: #FF0000 -> mapeia pra cor da skin
- Cor marcadora terno: #00FF00 -> mapeia pra cor da skin
- Cor marcadora veias: #0000FF -> mapeia pra cor da skin
- Etc.
Isso reduz de 55 sheets pra 11 sheets + 5 paletas.
