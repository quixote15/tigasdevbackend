## BATCH GENERATION NOTES

### For Stable Diffusion / ComfyUI
- **Model recommended:** Pixel art LoRA on SD 1.5 or SDXL
- **Sampler:** DPM++ 2M Karras
- **Steps:** 30-40
- **CFG Scale:** 7-9 (higher for more prompt adherence)
- **Resolution:** Generate at 256x256, downscale to 64x64 with nearest-neighbor (NO bilinear/bicubic -- preserves pixel edges)
- **ControlNet:** Use reference image of Andre Guedes cartoon style as style reference
- **Batch count:** Generate 8-12 per prompt, manually select best match

### For DALL-E 3 / Midjourney
- Add "pixel art, 64x64 sprite" prominently in prompt
- Add "--no realistic, photograph, 3D render" for Midjourney
- Use --style raw for Midjourney to avoid beautification
- Post-process: manual pixel cleanup, outline thickening, color dirtying in Aseprite

### Post-Processing Pipeline (ALL outputs)
1. Resize to 64x64 nearest-neighbor in Aseprite/Photoshop
2. Thicken outlines to 2-3px if AI generated them thinner
3. Dirty up any colors that look too clean (add noise, darken highlights)
4. Verify chin is DOMINANT feature -- if not, re-draw chin larger
5. Check belly scar visibility -- add if missing
6. Ensure forced grin looks UNNATURAL -- adjust if it looks genuine
7. Add/fix transparency around character edges
8. Test at 1x, 2x, and 4x zoom for pixel clarity
9. Assemble into horizontal sprite sheet strips
10. Validate against Phaser 3 frameWidth/frameHeight requirements
