## 17. Parametros de Geracao Recomendados

### Midjourney
- `--style raw` (essencial — evita estilizacao brilhante do MJ)
- `--ar 1:1` para tiles, `--ar 16:9` para cenas
- `--chaos 20-30` (menos caos que personagens — cenario precisa de consistencia)
- `--no bright, colorful, space, stars, daylight`
- `--s 100-150` (estilizacao moderada)

### Stable Diffusion
- **Sampler**: DPM++ 2M Karras
- **Steps**: 30-40
- **CFG**: 9-11 (mais alto — queremos aderencia forte ao void escuro)
- **Notas**: Modelos tendem a adicionar luz/brilho. Usar negative prompt AGRESSIVO contra brilho.
- **Inpainting**: Util para remover luz nao desejada apos geracao

### DALL-E
- Enfatizar "ABSOLUTE DARKNESS" e "nearly invisible" repetidamente
- DALL-E tende a iluminar demais — pedir "the darkest scene you can generate"
- Util para props individuais (mesa, mic, cadeira) mais que para cenario completo

## 18. Fluxo de Trabalho: Prompt → Pixel Art Tileset

### Etapa 1: Referencia de Ambiente
1. Gerar cenario completo com prompt #1 e #10
2. Extrair sensacao geral de "void com objetos flutuantes"
3. NAO copiar diretamente — usar como mood board

### Etapa 2: Props Individuais
1. Gerar cada prop separadamente (prompts #3-#9, #14)
2. Para cada prop gerado, fazer 2-3 variacoes
3. Escolher a que melhor funciona a 16x16px
4. Simplificar manualmente para pixel art

### Etapa 3: Texturas Base
1. Gerar void textures com prompt #2
2. Gerar cone de luz com prompt #12
3. Gerar glow com prompt #13
4. Gerar veias de energia com prompt #15
5. Adaptar para tiles 16x16 com dithering manual

### Etapa 4: Montagem
1. Todos os tiles desenhados MANUALMENTE em Aseprite
2. Arte gerada serve APENAS de referencia
3. Montar tileset 256x256 conforme layout da tileset-spec.md
4. Testar seamless em grid de teste
5. Montar mapa no Tiled Map Editor
6. Exportar JSON
7. Testar no Phaser 3

### Etapa 5: Validacao
1. O cenario parece um "podcast no void"?
2. E o OPOSTO visual da Esplanada?
3. 90%+ do mapa e escuridao?
4. Os props parecem flutuar?
5. O glow laranja e o unico calor visual?
6. O letreiro "LIMBO" e legivel e pisca?
7. A textura de papel esta aplicada?
8. Nenhuma cor e pura?

## Notas Importantes

- O Limbo deve parecer BARATO — nao e um inferno epico, e um estudio de podcast improvisado no nada. A precariedade e a piada.
- A ESCURIDAO e o principal "asset". A maioria dos tiles e variacao de preto. A arte esta no que FALTA, nao no que tem.
- O unico ponto de calor visual (laranja) e a cadeira do Monark — isso o coloca como centro gravitacional do cenario.
- O verde doentio do neon e das luzes conecta visualmente o Limbo ao gas da Esplanada (ambos `#3D6B3A`) — sutil referencia cruzada.
- Props quebrados contam uma historia: esse equipamento de podcast foi "cancelado" junto com os donos. Funcionava, nao funciona mais. Flutua no void como restos de naufrago.
- A estetica de Robert Crumb/Andre Guedes se aplica nos CONTORNOS GROSSOS dos props e no cross-hatching — o void em si e minimalista, mas os objetos nele sao grotescos.
