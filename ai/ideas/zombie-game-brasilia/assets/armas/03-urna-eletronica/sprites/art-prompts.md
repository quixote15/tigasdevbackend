## Post-Processing Notes
- Weapon frames: clean up to exact 48x48 pixel grid
- Projectile frames: clean up to exact 32x32 pixel grid
- The CRT green glow is CRITICAL -- it should look like actual green phosphor (#00FF41)
- Screen cracks must be visible but not obscure the screen content
- Exposed wires should have distinct colors (red, yellow, blue) for character
- "CONFIRMADO" text on screen should be chunky pixel font, not smooth
- The ballot paper projectile needs to read clearly at 32x32 -- keep it simple
- The red "CONFIRMADO" stamp on ballots is the key visual identifier
- Electrical effects (sparks, arcs) use ADD blend mode -- keep them bright
- Assemble weapon sheet: 528x48 (11 frames x 48px)
- Assemble projectile sheet: 128x32 (4 frames x 32px)
