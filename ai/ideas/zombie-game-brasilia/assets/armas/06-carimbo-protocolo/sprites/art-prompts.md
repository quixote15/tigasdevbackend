# Carimbo de Protocolo — Art Generation Prompts

> All prompts target 48x48 pixel art sprites with transparent background unless noted.
> Style anchor: Robert Crumb underground comix + Brazilian political satire + B-movie horror. Grotesque, exaggerated, dirty saturated colors. Thick outlines, heavy shadows. The stamp is COMICALLY OVERSIZED.

---

## Static / Inventory

### Frame 0 — Static Stamp
```
Pixel art, 48x48, transparent background. A comically oversized rubber stamp weapon viewed from top-down isometric angle. Battered, well-used bureaucratic weapon. Underground comix style, thick black outlines (2px), heavy shadows, dirty saturated colors. Brazilian political satire aesthetic.
```
