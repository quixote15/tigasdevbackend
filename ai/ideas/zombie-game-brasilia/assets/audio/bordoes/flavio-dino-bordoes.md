# FLAVIO DINO - Scripts de Bordoes para Audio/TTS

## Direcao Geral de Voz

- **Tipo**: Grave, burocratico, monotonico - cada palavra e um CARIMBO
- **Referencia**: Andre Guedes fazendo voz de burocrata robótico. Tipo funcionario publico que carimba documentos ha 40 anos e encontrou prazer existencial em negar coisas. Maquina de NEGAR
- **Pitch**: Muito grave, monotono, quase robótico
- **Velocidade**: Lenta e ritmica - cada palavra no mesmo tempo, como carimbo batendo em sequencia
- **Sotaque**: Maranhense leve, burocratico federal, tom de relatorio
- **Maneirismos**: Fala como se estivesse LENDO um despacho judicial. Monotono por padrao, mas a palavra "NEGADO" e SEMPRE gritada com prazer sadico - e o unico momento de emocao. Cada frase soa como se tivesse numero de protocolo. Pausas entre frases como quem troca de pagina
- **Respiracao**: Mecanica, regular, como maquina de carimbar

## Bordao 01: "NEGADO."

| Campo | Valor |
|---|---|
| **Texto exato** | "NEGADO." |
| **Emocao/Intensidade** | Poder burocratico absoluto, prazer sadico - 9/10 |
| **Duracao estimada** | 0.8s |
| **Direcao TTS** | UMA PALAVRA. Gritada. Seca. Definitiva. "NE-GA-DO" com cada silaba sendo uma martelada de carimbo. Grave no extremo. A palavra mais satisfatoria que este homem ja pronunciou na vida. Nao ha explicacao, nao ha apelacao. Apenas NEGADO. O silencio apos a palavra deve durar 0.3s - o eco do carimbo. E a unica palavra que ele fala com EMOCAO. |
| **Trigger no jogo** | Ataque basico com carimbo |
| **Prioridade** | 1 - toca sempre |
| **Efeitos sonoros** | CARIMBO GIGANTE batendo (som pesado, metalico). Tinta vermelha espirando. Papel amassando sob o impacto. Eco de reparticao publica vazia. |

## Bordao 04: "Emenda? Que emenda? NEGADA."

| Campo | Valor |
|---|---|
| **Texto exato** | "Emenda? Que emenda? NEGADA." |
| **Emocao/Intensidade** | Deboche burocratico, negacao com prazer - 8/10 |
| **Duracao estimada** | 2.0s |
| **Direcao TTS** | "Emenda?" como se tivesse ouvido uma palavra absurda pela primeira vez. Tom de incredulidade falsa. "Que emenda?" repetindo a pergunta com mais desprezo, como se o conceito nao existisse. "NEGADA" - MARTELADA FINAL. Gritada com o mesmo prazer sadico de sempre. O ritmo e: confusao -> desprezo -> DESTRUICAO. Cada fase mais grave que a anterior. |
| **Trigger no jogo** | Ao bloquear orcamento inimigo (contra-ataque / neutralizar buff) |
| **Prioridade** | 1 - toca sempre |
| **Efeitos sonoros** | Som de papel sendo rasgado ao meio na palavra "NEGADA". Carimbo vermelho gigante. Pedacos de emenda parlamentar voando como confete. Gaveta de arquivo batendo. |

| **Emocao/Intensidade** | Auto-revelacao comica, honestidade involuntaria - 6/10 |
| **Duracao estimada** | 4.0s |
| **Direcao TTS** | TRES FASES. "Eu nao bloqueio por prazer" em tom serio, formal, burocratico, como quem le uma declaracao oficial (0.0-1.5s). | **Prioridade** | 2 - toca frequentemente |
| **Efeitos sonoros** | Maquina de escrever ao fundo nos dois primeiros trechos. Silencio no terceiro. Leve risadinha nasal abafada no final. Carimbo fantasma (som distante). |

## Nota Especial de Producao

Flavio Dino e o boss mais RITMICO do jogo. Suas falas devem ser mixadas com timing de CARIMBO - cada silaba no mesmo intervalo, como maquina de carimbar automatica. O efeito sonoro do carimbo e tao parte do personagem quanto a voz. Toda fala deve ter ao menos um som de carimbo acompanhando.

A excecao e a palavra "NEGADO" que aparece em varias falas. Sempre que "NEGADO/NEGADA" aparece, deve ser tratada como evento sonoro especial: volume +3dB, carimbo gigante, tela tremendo levemente. E a palavra de poder dele - o equivalente ao "Monocraticamente" do Xandao.

## Direcao de Mixagem

- **Volume base**: Normal, mas monotono (pouca variacao dinamica)
- **Volume de "NEGADO"**: +3dB acima do resto, com compressao para impacto
- **Efeito base**: Reverb de reparticao publica (eco frio, fluorescente)
- **Ritmo**: BPM constante nas falas - como maquina de escrever. 90-100 BPM equivalente
- **Layering**: Sempre camada de carimbo sob a voz. O carimbo e o "instrumento" dele
