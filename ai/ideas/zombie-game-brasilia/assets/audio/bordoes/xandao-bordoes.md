# XANDAO (Alexandre de Moraes) - Scripts de Bordoes para Audio/TTS

## Direcao Geral de Voz

- **Tipo**: Grave, autoritario, intimidador
- **Referencia**: Andre Guedes fazendo voz de delegado/juiz truculento
- **Pitch**: Muito baixo (baryton profundo)
- **Velocidade**: Lenta e pausada - cada palavra e uma SENTENCA
- **Sotaque**: Paulistano carregado, confiante
- **Maneirismos**: Fala como se estivesse dando sentenca judicial. Cada silaba e martelada. Pausas dramaticas entre palavras. Nunca grita - a intimidacao vem do tom grave e do peso das palavras
- **Respiracao**: Pesada, controlada, como um touro antes de atacar
- **TTS Base**: `pt-BR-AntonioNeural` — pitch -5Hz, rate -10% para gravidade maxima

## Efeito Sonoro Especial: Sequencia de Censura

Quando Xandao censura, sequencia completa de audio:
1. MARTELADA grave (gavel slam)
2. Static/ruido branco (0.5s)
3. SILENCIO absoluto (2s - nem musica, nem SFX)
4. "Censurado!" (voz do Xandao)
5. Som retorna gradualmente
