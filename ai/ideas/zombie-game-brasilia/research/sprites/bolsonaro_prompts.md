## NOTAS TÉCNICAS PARA GERAÇÃO

- **Resolução**: Gerar em 1024x1024 e depois recortar/redimensionar
- **Sprite sheets**: DALL-E não faz sprite sheets perfeitos — gere cada frame como imagem separada e monte no Aseprite/TexturePacker
- **Consistência**: Use "seed" se disponível para manter o mesmo personagem entre gerações
- **Pós-produção**: Remover fundo, ajustar paleta para consistência, alinhar frames
- **Alternativa**: Use os prompts como base no Midjourney (adicionar `--style raw --ar 1:1`) ou Stable Diffusion (usar LoRA de cartoon/comix)
